# universal/blockers

Rules that apply cross-language. Load this when working with multiple languages or when the `base.md` for a language is unavailable.

- `S2068` — Never hard-code credentials, passwords, tokens, API keys
- `S2755` — Never parse XML without disabling external entity resolution
- `S1451` — All source files SHALL have copyright/license headers
- `S1313` — Never hard-code IP addresses
- `S2077` — Never format SQL queries by concatenating user input
- `S4721` — Never execute OS commands with shell interpreter
