# Skills Library

## Purpose

- Reusable skills for code agents.
- Distribute standards, references, and workflows as isolated, per-skill packages.

## Technologies

- Angular 21
- Vue 3
- React / Next.js
- TypeScript
- Markdown with `SKILL.md` and `references/`

## Installation

```bash
npx skills add https://github.com/hebertpaziam/skills
```
